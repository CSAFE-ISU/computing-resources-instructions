Chapter 9 Pronto Commands

9.1 List loaded modules

  1. In an interactive session, type

    $ module list

9.2 Purge all loaded modules

  1. In an interactive session, type

    $ module purge

9.3 View your job(s) in the Pronto queue

  1. In pronto terminal (not in an interactive session), type

    $ squeue -u your-netid

9.4 Cancel a job

  1. In pronto terminal (not in an interactive session), type

    $ scancel jobid

    where jobid is the six-digit number for the job you want to cancel.
